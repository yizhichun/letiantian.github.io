试用Atom编辑器

发表于 May 10th 2014 | |

2014-05-10

Atom是github出品的一款编辑器，还不算完善，从界面上看和sublime类似，目测会对sublime造成很大压力。

安装

当前(2014-05-10)官网仅提供了mac安装版本（67M，真大～），Atom项目中readme.md也提供了在windows、linux平台构建Atom的方法。如果有兴趣可以手动构建。

在ubtuntu（必须是64位）下也可以通过PPA安装Atom，该安装方法来自Install Atom in Ubuntu (64bit) via PPA：
依次敲入以下命令即可：

sudo add-apt-repository ppa:webupd8team/atom
sudo apt-get update
sudo apt-get install atom

Atom在linux对中文支持良好。

如何设置

依次打开File -> Preferences，就会出现设置窗口，使用者可以根据爱好进行自定义，同时设置窗口也提供了插件搜索和安装功能。

几个快捷键

ctrl-p -> 打开文件搜索面板
ctrl-Shift-P -> 打开名命令面板

几个插件

autocomplete + -> 不使用快捷键进行自动补全
Highlight Line -> 高亮当前行
Markdown Preview -> 预览markdown（快捷键ctrl-shift-M即可预览）
Markdown Stream -> 在浏览器中实时预览markdown文件

关于自动换行

Preference -> Settings -> 选中Soft Wrap

或者

View -> Toggle Soft Wrap

或者在配置文件config.cson中添加：

'editor':
  'softWrap': true

暂时就这些了。
